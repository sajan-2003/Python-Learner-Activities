# Maze Adventure Game (Python)

A simple text-based maze game built using Python.
The player navigates through a maze to reach the treasure while avoiding blocked paths.

## Controls

* **W** – Move Up
* **S** – Move Down
* **A** – Move Left
* **D** – Move Right

## Features

* Grid-based maze
* Player movement logic
* Win condition
* Beginner-friendly code structure

## How to Run

```bash
python maze_game.py
```

## Learning Outcomes

* 2D lists
* Functions
* Loops
* Conditional logic
* Game logic basics
